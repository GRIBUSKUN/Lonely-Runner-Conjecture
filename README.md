# Lonely Runner Conjecture Visualization Project

## Overview
The Lonely Runner Conjecture is a fascinating problem in number theory that revolves around the distribution of runners on a circular track. According to this conjecture, if there are n runners with different constant speeds, there exists a time at which each runner is at least a distance of 1 unit away from each other on the circumference of the track.

This visualization project aims to provide an interactive and intuitive representation of the Lonely Runner Conjecture, allowing users to explore how runners move and how their positions relate to one another over time.

## Features
- **Dynamic Visualization**: Visual display of runners on a circular track.
- **Adjustable Parameters**: Users can modify the number of runners and their speeds.
- **Real-time Simulation**: Observe the immediate effects of parameter changes.
- **Statistical Analysis**: View statistical data concerning runner distances and the conjecture's validity.

## Getting Started
### Prerequisites
- A modern web browser (Chrome, Firefox, etc.)
- Familiarity with HTML, CSS, and JavaScript for understanding the code.
### For development only
- NodeJs

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/GRIBUSKUN/Lonely-Runner-Conjecture.git
   cd Lonely-Runner-Conjecture
   ```
### For simple initialization
2. Open `index.html` in your web browser.
## For development
2. Install the require npm packages
   ```bash
   npm install
   npm run dev
   ```
## Usage
- Use the sliders to adjust the number of runners and their speeds.
- Click "Start Simulation" to begin the visualization.
- Observe the positions of runners change over time, and analyze the distances between them.
